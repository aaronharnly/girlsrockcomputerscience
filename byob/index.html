Installing BYOB

We will be using BYOB to learn programming. Please follow the instructions below to install BYOB on your laptops.

  1. Click here and BOYB download will begin
  2. Open Finder from the Dock
  3. Click Go from the top of the menu and select Downloads
  4. Double click the file BYOB 3.1.1 Mac.zip
  5. Right-click on the file BYOB 3.1.1 Mac and select Copy BYOB 3.1.1 Mac
  6. Right click on the Desktop and click Paste Item
